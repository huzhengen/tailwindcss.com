# Tailwind CSS 文档

Tailwind CSS 文档使用 [Next.js](https://nextjs.org/)。下面介绍如何在本地运行项目：

1. 克隆该仓库

    ```sh
    git clone https://github.com/tailwindlabs/tailwindcss.com.git
    ```

2. 进入项目根目录

    ```sh
    cd tailwindcss.com
    ```

3. 安装 JS 依赖

    ```sh
    yarn
    ```

4. 启动开发服务器

    ```sh
    yarn dev
    ```

另外，你也可以使用 `yarn export` 命令生成文档的静态导出。导出将保存在 `out` 文件夹中。
